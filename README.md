# Slot DeBot

Easy slot machine writen on smart contracts.

### About deployment

#### deploy locally

Add surf1 signer (your surf wallet) to the signers list and run:

```bash
./l.sh # local
```

#### deploy in dev network

```bash
./d.sh # dev
```
